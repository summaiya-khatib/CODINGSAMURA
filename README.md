# CODINGSAMURA
Sales Data Analysis project using Excel. Includes data cleaning, basic statistics, and visualization using pivot tables, bar charts, and pie charts to derive business insights.

Sales Data Analysis (Excel Project)

Overview
This project focuses on analyzing retail sales data using Microsoft Excel. The goal is to perform basic data analysis and create visualizations to derive meaningful insights.

Objectives
- Analyze sales performance across categories and regions
- Apply basic statistical techniques
- Create visual dashboards for better understanding

Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar Chart, Pie Chart)

Dataset
- Retail sales dataset (Superstore dataset)
- Contains information about sales, profit, category, and region

Analysis Performed
- Data Cleaning
- Basic Statistics (Total Sales, Profit, Average Sales)
- Sales by Category
- Sales Distribution (Pie Chart)
- Sales by Region

Dashboard Features
- Bar Chart: Sales by Category
- Pie Chart: Sales Distribution
- Interactive Slicers (Category, Region)

 Key Insights
- Technology category has the highest sales
- Sales are distributed across categories
- Some regions perform better than others

Files Included
- Excel Dashboard File (.xlsx)
- Project Report (optional)

 Conclusion
This project demonstrates basic data analytics skills including data cleaning, statistical analysis, and visualization using Excel
